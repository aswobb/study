## 本课重点

这一课主要练习一般疑问句。一般疑问句就是可以用 `Yes` 或 `No` 回答的问题。

```text
Are you busy?
Do you like me?
Does he like me?
```

一般疑问句主要分两类。

## 句型一：be动词提前

如果句子里有 `am / is / are`，变成疑问句时，把 `am / is / are` 放到句子前面。

```text
This apple is very big.
Is this apple very big?
```

```text
These workers are very busy.
Are these workers very busy?
```

回答方式：

```text
Yes, it is.
No, it isn't.

Yes, they are.
No, they aren't.
```

## 句型二：用 Do / Does 开头

如果句子里没有 `am / is / are`，而是普通动词，比如 `like / sit / drink`，就用 `Do` 或 `Does` 开头。

```text
You like me.
Do you like me?
```

```text
He likes me.
Does he like me?
```

注意：用了 `Does` 以后，后面的动词要变回原形。

```text
He likes me.
Does he like me?
```

不能写成：

```text
Does he likes me?
```

## Do 和 Does 的区别

```text
I / you / we / they + do
he / she / it / 单数名词 + does
```

例如：

```text
Do you drink water?
Does he sit down?
Does she like apples?
Does this dog eat fish?
```
